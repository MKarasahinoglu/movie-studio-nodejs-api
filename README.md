# Movie Studio API
A simple api for users to create their own movie library.
## Technologies Used
- Node.js
- Express.js
- Moongose
- JWT
## User Operations
- Register
- Login
- Logout
- Add a movie
- Update a movie
- Delete a movie
- List a movie by id
- List all movies
### Features
- JWT based authentication
- User-specific movie-related operations
### Dependencies
- cookie-parser
- dotenv
- express
- jsonwebtoken
- mongoose
#### Example .env file
```javascript
DATABASE_URI="uri"
DATABASE_NAME="name"
PORT=portNumber
ACCESS_TOKEN_KEY="Key"
REFRESH_TOKEN_KEY="Key"
ACCESS_TOKEN_EXPIRE_TIME="time"
REFRESH_TOKEN_EXPIRE_TIME="time"
```
