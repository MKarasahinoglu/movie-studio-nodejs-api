# Movie Studio API
A simple api for users to create their own movie library.
## Technologies Used
- Node.js
- Express.js
- Moongose
- JWT
## User Operations
- Register
- Login
- Logout
- Add a movie
- Update a movie
- Delete a movie
- List a movie by id
- List all movies
### Features
- JWT based authentication
- User-specific movie-related operations
